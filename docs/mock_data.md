# Mock data

The project includes a sample of mock data in a `csv` file generated with [Mockaroo](https://www.mockaroo.com/).

| Field name      | Data type (Mockaroo) |
|-----------------|----------------------|
| id              | Row Number           |
| first_name      | First Name           |
| last_name       | Last Name            |
| language        | Language             |
| gender          | Gender               |
| city            | City                 |
| job_title       | Job Title            |
| university      | University           |
| a_boolean_field | Boolean              |
